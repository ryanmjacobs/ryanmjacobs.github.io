Cloning Websites - October 13, 2014

How to Do It

wget -mpk http://ryanmjacobs.github.io/

That commmand right there will clone my website locally. Now to view it, all you have to do is open ./ryanmjacobs.github.io/index.html in your browser. Typically you can just run:

google-chrome ryanmjacobs.github.io/index.html

or if you're on Firefox

firefox ryanmjacobs.github.io/index.html

or if you're on some other obscure browser just point it to the location of the file.

Another Example

This will clone the contents of example.org.

wget -mpk http://example.org

Options Breakdown

Option Purpose
-m, --mirror turn on options suitable for mirroring
-p download all files necessary to display page, e.g. CSS, images, etc.
-k, --convert convert links to reference the local copies, not the online versions
