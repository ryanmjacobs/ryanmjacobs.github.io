Installing darkhttpd - October 09, 2014

"When you need a web server in a hurry."

darkhttpd is a great, lightweight web server by Emil Mikulic. However, many Linux distros don't offer a package for it. This guide will go through all of the steps of obtaining and installing this amazing program.

Debian/Ubuntu

# Download and install build tools, (if you don't already have them)
sudo apt-get update
sudo apt-get install build-essential
# Clone repository, generate debian package, and install!
git clone https://github.com/ryanmjacobs/darkhttpd # clone the repository
cd darkhttpd                                       # go into repository
make debian                                        # generate debian package
sudo dpkg -i darkhttpd.deb                         # install debian package

Invocation & Usage

To start the web server, go to the directory that contains all of you web files, (.html, .cs, .js, etc.) and run 'darkhttpd .'. Notice that's 'darkhttpd DOT', but with a real dot. In BASH, the dot represents your current directory. Navigate your browser to localhost:8080 to see what's being hosted.

ryan@delta ~ $ cd working/GitHub/site/
ryan@delta ~/working/GitHub/site $ darkhttpd .
darkhttpd/1.10, copyright (c) 2003-2014 Emil Mikulic.
listening on: http://0.0.0.0:8080/
^C^C^C
ryan@delta ~/working/GitHub/site $

BEST EXAMPLE EVER!

What better better way to demonstrate the usage of darkhttpd than to use it to host the website you are currently viewing? In fact, I do it all the time before I push changes to the site! 

git clone https://github.com/ryanmjacobs/ryanmjacobs.github.io
darkhttpd ryanmjacobs.github.io --port 1234

Now point your browser to localhost:1234! And, browse my site locally! No internet connection required :D
