The Art of Shell Scripting - October 03, 2014

What is a Shell Script?

A shell script is simple a list of commands to be executed by the shell. It can include statements such as if, else, while, and for.

Let's Create One!

First let's create the file. The file extension is typically .sh, but as we all know, "File extensions don't matter in Linux."

ryan@delta ~/example $ touch myScript.sh
ryan@delta ~/example $ ls
myScript.sh
ryan@delta ~/example $ chmod +x myScript.sh 
ryan@delta ~/example $ ls
myScript.sh

Let's break it down:

  • 'touch myScript.sh' creates the file.
  • 'chmod +x myScript.sh' makes the file executable, (we can now run it with ./myScript.sh).

Start it off with a bang!

Or more precisely, a shebang, also called a sha-bang, hashbang, or hash-exclam. It looks like this: #!.

Each interpreted language has it's own shebang to identify the file. BASH scripts uses the following shebang, which is what we'll be using.

#!/bin/bash

Echo, Echo, Echo

Type this code into your script:

#!/bin/bash

# This file echos out some random information.
# Note: BASH interprets '#' signs as comments.

# Begin the echo process :)
echo "Initializing hardcore echo routines..."
sleep 1
echo "done!"

# Echo out some cool numbers
echo ""
echo "Some cool numbers:"
echo "    pi   -> 3.14"
echo "    life -> 42"

# Quit
echo ""
echo "Quiting in 3..."
sleep 1
echo "Quiting in 2..."
sleep 1
echo "Quiting in 1..."
sleep 1
echo "Quiting NOW!"

Now run it! If all goes well, you should get something like this:

ryan@delta ~/example $ ./myScript.sh 
Initializing hardcore echo routines...
done!

Some cool numbers:
    pi   -> 3.14
    life -> 42

Quiting in 3...
Quiting in 2...
Quiting in 1...
Quiting NOW!

Variables

Variables are declared with an '='. They are untyped. To reference one, use a '$'. 

var1="Hello World"
var2=42

echo "$var1, I love the number $var2!"

Conditional Statements & Loops

Okay, now we're going to whip through BASH's conditional statements and loops.

If/Else Statement

# -eq, -ne, -gt, -lt, -ge, -le are for numbers
if [ $num1 -eq $num2 ]; then
    echo "num1 is equal to num2"
elif [ $num1 -gt $num2 ]; then
    echo "num1 is greater than num2"
else
    echo "num1 is not equal to num2"
fi

# ==, != are for strings
if [ $str2 == $str2 ]; then
    echo "str1 is equal to str2"
fi

While Loop

while true; do
    echo "flood my terminal"
done

i=0
while [ $i -lt 100 ]; do
    let i++
    echo $i
done

For Loop

for i in {0..100}; do
    echo $i
done

array=('I' 'Am' 'An' 'Array')
for word in ${array[@]}; do
    echo $word
done
