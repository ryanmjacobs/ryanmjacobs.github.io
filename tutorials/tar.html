tar - October 02, 2014

Tape Archive

The tar program is used to create, maintain, modify, and extract files that are archived in the tar format. One odd thing about the tar command is that the options are expressed without dashes, unlike typical Unix commands.

Command Synopsis

tar [options] [-f file] [directory]

Compression Method

Option Purpose
z gzip compression
j bzip2 compression
J xz compression

Action

Option Purpose
x extract
c compress
t list contents, (don't extract)

Other

Option Purpose
v be verbose
f file will be next argument

Learning By Example

# Extracting .tar.gz files
tar zxvf archive.tar.gz

# Creating .tar.gz files
tar zcvf archive.tar.gz folderToCompress/
